# Library Management System (Java)

## Files
- src/Book.java
- src/User.java
- src/Library.java
- src/Main.java

## Compile & Run
```bash
javac src/*.java
java -cp src Main
```

## Features
- Add/list/search books
- Add users
- Issue and return books with borrow limits
- Simple interactive CLI
- Demonstrates OOP: abstraction, encapsulation, method overriding, polymorphism
